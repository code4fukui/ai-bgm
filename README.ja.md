# ai-bgm

AI生成BGM。

## デモ
https://code4fukui.github.io/ai-bgm/

## 機能
- Magenta MusicVAEモデルを使用してBGMを生成
- テンポ、ビート、長さをカスタマイズ可能
- Splendid Grand Pianoライブラリを使用したピアノ音源

## 要件
- Web Audio APIをサポートするWebブラウザ

## 使い方
1. [デモページ](https://code4fukui.github.io/ai-bgm/)を開く
2. 必要に応じてテンポ、ビート、長さを調整する
3. 「play」ボタンをクリックしてBGMを生成・再生する

## 依存関係
- [smplr](https://github.com/code4fukui/smplr)
- [Magenta](https://github.com/magenta/magenta)
