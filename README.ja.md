# AI-BGM

AIで生成された背景音楽を再生できるWebアプリです。Magentaの音楽生成モデルを使用しており、テンポ、拍子、長さなどを指定して好みの曲を生成できます。

## 機能
- テンポ、拍子、長さの設定
- Magentaの音楽生成モデルによる自動作曲
- ピアノ演奏による再生

## 必要環境
特に必要ありません。ブラウザで動作します。

## 使い方
1. テンポ、拍子、長さを設定します。
2. 「play」ボタンをクリックすると、AIが生成した曲が再生されます。
3. 再生を停止するには「play」ボタンをもう一度クリックします。
