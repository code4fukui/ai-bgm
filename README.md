# ai-bgm

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A web application that generates background music using Magenta's MusicVAE model.

## Demo

https://code4fukui.github.io/ai-bgm/

## Features

- Generates AI-powered melodies using the [Magenta MusicVAE](https://magenta.tensorflow.org/music-vae) model.
- Customize the tempo, beat, and length of the generated music.
- Renders audio using the Splendid Grand Piano sound from the [smplr](https://github.com/code4fukui/smplr) library.

## Requirements

- A modern web browser with support for the Web Audio API.

## Usage

1.  Open the [demo page](https://code4fukui.github.io/ai-bgm/).
2.  Adjust the tempo, beat, and length settings to your preference.
3.  Click the "play" button to generate and listen to the music.

## Built With

-   [Magenta.js](https://github.com/magenta/magenta) - A JavaScript library for music and art generation.
-   [TensorFlow.js](https://www.tensorflow.org/js) - The underlying machine learning framework.
-   [smplr](https://github.com/code4fukui/smplr) - A simple Web Audio API sampler for instrument sounds.
