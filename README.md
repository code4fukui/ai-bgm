# ai-bgm

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

AI-generated background music.

## Demo
https://code4fukui.github.io/ai-bgm/

## Features
- Generates background music using Magenta MusicVAE model
- Customizable tempo, beat, and length
- Piano sound using the Splendid Grand Piano library

## Requirements
- Web browser with support for Web Audio API

## Usage
1. Open the [demo page](https://code4fukui.github.io/ai-bgm/)
2. Adjust the tempo, beat, and length as desired
3. Click the "play" button to generate and play the background music

## Dependencies
- [smplr](https://github.com/code4fukui/smplr)
- [Magenta](https://github.com/magenta/magenta)
